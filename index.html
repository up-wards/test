Git / GitHub Notes

1. Основные команды

  • git init — создать новый локальный репозиторий
  • git add . — добавить все изменения в индекс
  • git commit -m "сообщение" — зафиксировать изменения

2. Работа с ветками

  • git branch <branch> — создать новую ветку
  • git checkout <branch> — перейти на ветку
  • git merge <branch> — смёржить ветку в текущую

3. Удалённый репозиторий

  • git remote add origin <url> — добавить удалённый репозиторий
  • git push -u origin main — отправить изменения
  • git pull — получить изменения с удалённого репозитория
